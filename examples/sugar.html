Pikaday + Sugar

A refreshing JavaScript Datepicker — lightweight, no dependencies, modular CSS.

Pikaday can be used with custom date parser and date formatter functions. This example uses the Sugar Javascript library instead of Moment.js to support natural language parsing (eg 'last wednesday', '2 weeks from monday, 'the end of next month', '1 year from now', '3 months ago') as well as many standard date formats. See Sugar's Dates page for more examples.
