Pikaday - Custom date format example

A refreshing JavaScript Datepicker — lightweight, no dependencies, modular CSS.

The dateFormatFn and dateParseFn callbacks can be used to apply custom formatting and parsing of the selected date.
This can be used to control the value written into the field which Pikaday is bound to.
This is also useful if you want to integrate a 3rd party parser or formatter eg Sugar JS - Dates, Fecha, etc.



What is this?

Since version 1.0 Pikaday is a stable and battle tested date-picker. Feel free to use it however you like but please report any bugs or feature requests to the GitHub issue tracker, thanks!
